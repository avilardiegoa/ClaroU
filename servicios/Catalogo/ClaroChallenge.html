Claro Challenge es un recurso diseñado para enseñar el uso de herramientas de Claro mediante la práctica guiada. Combina instrucciones paso a paso con una interfaz simulada donde el usuario debe ejecutar acciones reales.

Cada recurso se estructura en hasta 4 subprocesos, cada uno convertido en una “misión” con un objetivo concreto. Cada subproceso debe incluir entre 2 y 5 pasos (slides), permitiendo un máximo excepcional de 6 cuando la tarea lo requiera. Tras completar los pasos, el usuario responde una pregunta-ficha que valida los conocimientos adquiridos. Cada pantalla debe mantener instrucciones breves (máx. 60 palabras) y no más de 3 elementos arrastrables. El tiempo estimado por subproceso es de hasta 3 minutos.

Regla clave

Paso correcto = avance asegurado.
Para diseñar retos claros y efectivos, consulta el manual de uso de Claro Challenge.
